Failure Type Prediction for Time Series sensor data

The project is developed to detect changes in events and for detecting anomalies in the time series data. The LSTM auto encoder decoder network is used as an unsupervised learning algorithm and create embeddings. The sensors contributing to the change are computed by using EI (Expandability Index) measure.

  • DDesign and implementation of automating feature selection and normalization using MinMax or Local mean normalization.
  • The model was able to detect anomalies and types of anomalies.
  • Implementing Multi-Layer LSTM auto encoder and Decoders for learning the model
  • Encoder embeddings are used for Detection of changes in the sensor data
  • For detecting type of anomalies LSTM classifier is used
  • Python libraries such as Pandas, Scikit-Learn, Keras Deep Learning Models are used

    Speech Recognition using Deep Learning algorithms

    The project consist of developing deep learning algorithms that perform recognition of iso- lated speech signals. We used Mel-Frequency Cepstral Coefficients (MFCC) as feature de- scriptors and a deep convolutional neural network for classification of speech signals. The project is implemented in Python using open source deep learning library (Keras) and ma- chine learning (SciKit) libraries.

  • Designing convolutional neural network algorithms for speech recognition
  • Extracting FFT and MFCC features from speech signals
  • Training and classifying speech signals using FFT and MFCC features.
  • Tested with multiple regional languages viz. English, Kannada, Telugu and Hindi audio datasets which are recorded under constrained studio settings.
  • Recognition accuracy of 95% is acheived by using MFCC features + DCNN as compared to FFT + DCNN on English language isolated speech samples.
