Optimizing Automatic Indexing of Audio books using Apache Hadoop Framework

The project is developed to automate the process of converting speech into text informa- tion from a large collection of English audio books and indexing these audio books based on converted text information. A cascaded alternate sequence of Mappers and Reducers is designed and implemented in order to exploit parallelism inherent within the modules of con- verting speech into text information and indexing of large collection of audio books based on extracted text.

  • Extracting text information from audio books using CMU SPHINX Speech to Text Con- verter
  • Indexing audio books using TF-IDF algorithm.
  • Implementing cascaded Hadoop Map-Reduce framework to exploit concurrency present within indexing of audio books.
  • Tested the perfomance of MapReduce based audio books indexing algorithm on a col- lection of around 500 audio books.

    Near Real time Face Recognition on Apache Spark Framework

    The project developed for detecting and recognizing multiple faces in real-time. Face feature extraction and classification algorithms are optimized in real-time by designing and imple- menting RDD based algorithms on Spark cluster.

  • Designing Spark RDD algorithms that performs extraction of image features and recog- nition of faces in real time
  • Optimization of face feature extraction and classification algorithms
  • Classification and Recognition of faces by using Support Vector Machine
  • Optimized LBP within Map and Reduce Phase
  • Face feature extraction and recognition algorithms are deployed and tested on an In- house cluster of 200 nodes
  • Computational speed up of around 5 frames per second
